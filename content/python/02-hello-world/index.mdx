## The Most Famous Program Ever

There's a tradition in programming: your first program should print the words "Hello, World!" to the screen.

Why? Because it proves:
- Your computer can run the language
- You can write and execute code
- You're officially a programmer!

Let's do it! Here's your first Python program:

```python
print("Hello, World!")
```

That's it! Just one line! When you run this, you'll see:

<TerminalDemo>
{`$ python3 hello.py
Hello, World!`}
</TerminalDemo>

<FunFact>
  The first "Hello, World!" program was written by Brian Kernighan in 1972 for the C programming language. Now it's how EVERY programmer starts!
</FunFact>

## Understanding print()

The `print()` function is your way to talk to the user! It displays text on the screen.

Let's break down the parts:

```python
print("Hello, World!")
#  ‚îÇ      ‚îÇ
#  ‚îÇ      ‚îî‚îÄ The message (in quotes)
#  ‚îî‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ‚îÄ The function name
```

**Important parts:**
- `print` - The function that displays text
- `()` - Parentheses that hold what you want to print
- `""` - Quotes around your text (called a "string")

## Printing Multiple Things

Want to print more than one thing on a line? Just separate them with commas:

```python
print("Hello", "there", "friend!")
```

Output:
<TerminalDemo>
{`Hello there friend!`}
</TerminalDemo>

Python automatically adds spaces between items!

## Print Numbers Too!

You don't need quotes for numbers:

```python

print(100)
print(2024)
```

<TerminalDemo>
{`42
100
2024`}
</TerminalDemo>

You can mix text and numbers:

```python
print("I am", 12, "years old")
print("My favorite number is", 7)
```

<TerminalDemo>
{`I am 12 years old
My favorite number is 7`}
</TerminalDemo>

## Print Blank Lines

Want to add space between your output? Just use print() with nothing inside:


## Common Mistakes (And How to Fix Them!)

Everyone makes mistakes when learning! Here are the most common:

### Mistake 1: Forgetting Quotes
```python
# Wrong:
print(Hello)

# Right:
print("Hello")
```

### Mistake 2: Forgetting Parentheses
```python
# Wrong:
print "Hello"

# Right:
print("Hello")
```

### Mistake 3: Mismatched Quotes
```python
# Wrong:
print("Hello')

# Right:
print("Hello")
# or
print('Hello')
```

<FunFact>
  In Python, you can use "double quotes" or 'single quotes' for text. Both work exactly the same! Just make sure they match on both ends.
</FunFact>

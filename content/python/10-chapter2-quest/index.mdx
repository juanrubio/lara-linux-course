## The Quest Mission

Build a Number Guessing Game with these features:

1. The secret number is between 1 and 20
2. The player gets multiple guesses
3. After each guess, give hints: "Too high!", "Too low!", or "Correct!"
4. Keep track of how many guesses they've made
5. Make it fun with colorful messages!

<FunFact>
  Number guessing games are a classic! They teach the concept of "binary search" - a super efficient way to find things that programmers use all the time!
</FunFact>

## Complete Game Code

Here's the complete game putting it all together! Type this into a Python file:

<TerminalDemo>
{`# Number Guessing Game - Complete Version
print("=" * 40)
print("üéÆ WELCOME TO THE GUESSING GAME! üéÆ")
print("=" * 40)
print("\\nI'm thinking of a number between 1 and 20.")
print("You have 3 chances to guess it!\\n")

# Set up the game
secret_number = 7
guess_count = 0
max_guesses = 3
won = False

# Guess 1
guess = int(input("Guess #1: "))
guess_count = 1

if guess == secret_number:
    print("üéâ AMAZING! You got it on the first try!")
    won = True
elif guess > secret_number:
    print("üìâ Too high! Try again.")
else:
    print("üìà Too low! Try again.")

# Guess 2 (only if first guess was wrong)
if not won and guess_count < max_guesses:
    guess = int(input("\\nGuess #2: "))
    guess_count = 2

    if guess == secret_number:
        print("üéâ CORRECT! You won in " + str(guess_count) + " guesses!")
        won = True
    elif guess > secret_number:
        print("üìâ Too high! One more try!")
    else:
        print("üìà Too low! One more try!")

# Guess 3 (only if still haven't won)
if not won and guess_count < max_guesses:
    guess = int(input("\\nGuess #3 (last chance!): "))
    guess_count = 3

    if guess == secret_number:
        print("üéâ CORRECT! You won in " + str(guess_count) + " guesses!")
        won = True
    else:
        print("‚ùå Sorry, you're out of guesses!")
        print("The secret number was " + str(secret_number))

# Final message
print("\\n" + "=" * 40)
if won:
    print("üèÜ VICTORY! You're a guessing master!")
else:
    print("Better luck next time!")
print("=" * 40)`}
</TerminalDemo>
